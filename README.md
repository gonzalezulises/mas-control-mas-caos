# Más control, más caos

> Límites externos para sistemas que no se detienen solos

Libro sobre por qué más control genera más caos en las organizaciones y cómo construir capacidades de auto-limitación.

**Autor:** Ulises González
**Editorial:** Rizo.ma
**Estado:** En desarrollo

## Estructura del repositorio

```
├── website/    # Landing page (mascontrolmascaos.com)
└── book/       # Manuscrito y herramientas de build
```

## Website

Landing page del libro desplegada en Vercel.

- **URL:** https://mascontrolmascaos.com
- **Stack:** HTML/CSS/JS estático con anime.js
- **Deploy:** Vercel (Root Directory: `website`)

## Book

Manuscrito del libro en Markdown con pipeline de generación a PDF/EPUB.

```bash
cd book
./build-book.sh full  # Genera PDF y EPUB
```

Ver `book/README.md` para documentación completa del manuscrito.

## Tesis Central

> Los sistemas organizacionales sanos renuncian deliberadamente a parte de su poder
> para preservar adaptabilidad, sentido y supervivencia.
